# 🚀 AI 编剧室 - 5分钟快速开始

## 第一步：确认服务器运行 (30秒)

开发服务器应该已经在运行。检查终端输出：

```
✓ Ready in 1636ms
- Local:   http://localhost:3000
- Network: http://192.168.1.193:3000
```

如果没有运行，执行：
```bash
cd /Users/daniel/Desktop/git/ai-movie-writer
npm run dev
```

---

## 第二步：打开浏览器 (10秒)

访问: **http://localhost:3000**

你应该看到：
- 🎬 大标题 "AI 编剧室"
- 📝 输入框 "电影主题"
- ▶️  "开始创作" 按钮

**悬疑类**:

### 操作步骤：

1. **输入主题** → 在输入框中输入上面任一主题
2. **点击"开始创作"** → 等待 AI 生成角色
3. **观看角色展示** → 3秒后自动开始创作
4. **实时观看** → 左侧显示剧本，右侧显示幕后花絮

## ❓ 遇到问题？

### 常见问题快速解决：

**Q: 页面打不开？**
```bash
# 检查服务器是否运行
ps aux | grep "next dev"

# 重启服务器
cd /Users/daniel/Desktop/git/ai-movie-writer
npm run dev
```

**Q: AI 没有响应？**
- 检查网络连接
- 检查 `.env.local` 中的 API 配置
- 查看浏览器控制台错误信息

**Q: 生成的内容不理想？**
- 尝试更具体的主题描述
- 重新生成（点击"重新开始"）
- 调整提示词（在 `app/actions.ts` 中）

## 💡 使用技巧

### 获得更好的剧本：

1. **主题要具体** ✅
   - ❌ "一个故事"
   - ✅ "2077年，黑客发现控制全人类的阴谋"

2. **包含冲突** ✅
   - ❌ "两个人聊天"
   - ✅ "两个敌对间谍被困在同一个房间"

3. **设定明确** ✅
   - ❌ "冒险"
   - ✅ "深海潜水员发现古代文明遗迹"

4. **角色关系** ✅
   - ❌ "一些人"
   - ✅ "父子两代侦探合作破案"
